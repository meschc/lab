# Генератор аватарок и имён

Локальный HTTP-сервис на Node.js + Express. Возвращает случайное имя (прилагательное + животное) и аватарку — абстрактный симметричный ASCII-паттерн с дизерингом и разноцветной палитрой, **отрендеренный в готовое изображение (SVG)**, чтобы клиенту не нужно было самому форматировать/раскрашивать ASCII.

## Запуск

```bash
npm install
npm run dev      # или npm start
```

Сервер стартует на порту `3000` (переопределяется через `PORT`).

## API

`GET /api/avatar?lang=ru|en`

| Параметр | Значения   | По умолчанию |
|----------|------------|--------------|
| `lang`   | `ru`, `en` | `en`         |

Ответ:

```json
{
  "name": "Swift Eagle",
  "img": "data:image/svg+xml;base64,PHN2ZyB4bWxucz...",
  "lang": "en"
}
```

- `img` — **готовое изображение** (SVG, 528×528, чёрный фон + цветные ASCII-символы) в виде data-URI. Вставляется напрямую в `<img src>` или сохраняется в файл. Клиенту не нужно форматировать ASCII самому.
- `lang` — язык сгенерированного имени (`ru` / `en`).
- Аватарка — абстрактный **симметричный** узор (метаболлы + кольцевая модуляция + дизеринг), симметрия по вертикали; палитра из 6 случайных цветов. Всё случайно при каждом запросе. Для русского языка род прилагательного согласуется с животным.

## Примеры curl

```bash
curl "http://localhost:3000/api/avatar?lang=ru"
curl "http://localhost:3000/api/avatar?lang=en"

# сохранить картинку в файл avatar.svg
curl -s "http://localhost:3000/api/avatar?lang=ru" \
  | node -e 'let s="";process.stdin.on("data",d=>s+=d).on("end",()=>{const u=JSON.parse(s).img;require("fs").writeFileSync("avatar.svg",Buffer.from(u.split(",")[1],"base64"))})'
```

## Демо-интерфейс в браузере

Откройте `http://localhost:3000/` — компактная демо-карточка:
- круглая иконка-аватар (на чёрном фоне) + имя;
- кнопки генерации **Новый · RU** / **Новый · EN** и круглая кнопка смены темы (светлая/тёмная, сохраняется в `localStorage`);

## Структура

```
/src
  server.js         — Express, роуты, демо-интерфейс
  nameGenerator.js  — словари имён, согласование рода (RU)
  asciiGenerator.js — абстрактный ASCII-паттерн + палитра + рендер в SVG
```

# Генератор аватарок и имён

Локальный HTTP-сервис на Node.js + Express. Возвращает случайное имя (прилагательное + животное) и аватарку — абстрактный симметричный ASCII-паттерн с дизерингом и разноцветной палитрой.

## Запуск

```bash
npm install
npm run dev      # или npm start
```

Сервер стартует на порту `3000` (переопределяется через `PORT`).

## API

`GET /api/avatar?lang=ru|en`

| Параметр | Значения   | По умолчанию |
|----------|------------|--------------|
| `lang`   | `ru`, `en` | `en`         |

Ответ:

```json
{
  "name": "Swift Eagle",
  "avatar_ascii": "  .:-=+**+=-:.  \n .-=+#%@@%#+=-. \n   ..:--:..    ",
  "colors": ["#3a7bd5", "#2fb3a0", "#6fd63a", "#d6c02f", "#d6742f", "#d62f4a"]
}
```

- `avatar_ascii` — многострочный ASCII-арт (сетка 44×26 символов), градации яркости через плотность символов (рамп ` .:-=+*#%@`).
- Аватарка — абстрактный **симметричный** узор (метаболлы + кольцевая модуляция + дизеринг), симметрия по вертикали.
- `colors` — палитра из 6 hex-цветов от светлого к тёмному. Клиент раскрашивает символ по его плотности: индекс символа в рампе → индекс в палитре. Для терминала можно игнорировать и показывать `avatar_ascii` как есть.
- Имя, узор и палитра случайны при каждом запросе. Для русского языка род прилагательного согласуется с животным.

## Примеры curl

```bash
curl "http://localhost:3000/api/avatar?lang=ru"
curl "http://localhost:3000/api/avatar?lang=en"

# показать саму картинку в терминале
curl -s "http://localhost:3000/api/avatar?lang=ru" \
  | node -e 'let s="";process.stdin.on("data",d=>s+=d).on("end",()=>console.log(JSON.parse(s).avatar_ascii))'
```

## Демо-интерфейс в браузере

Откройте `http://localhost:3000/` — компактная демо-карточка:
- круглая иконка-аватар (на чёрном фоне) + имя;
- кнопки генерации **Новый · RU** / **Новый · EN** и круглая кнопка смены темы (светлая/тёмная, сохраняется в `localStorage`);

## Структура

```
/src
  server.js         — Express, роуты, раскраска в браузерном просмотрщике
  nameGenerator.js  — словари имён, согласование рода (RU)
  asciiGenerator.js — абстрактный ASCII-паттерн + генерация палитры
```
